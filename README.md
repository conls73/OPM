# Ogden Pest Management - Modern Website

A beautiful, modern Next.js website for a solo pest control business in Central Oregon. Built to feel warm and trustworthy while showcasing professional services.

## 🚀 Features

- **Stunning Hero Section** - Eye-catching gradient with clear value proposition
- **Services Showcase** - Detailed service cards with pricing and features  
- **Trust Builders** - Testimonials and why-choose-us section
- **Service Areas** - Clear coverage map of Central Oregon
- **Mobile Responsive** - Beautiful on all devices
- **Contact Integration** - Easy ways to reach (phone, email, contact form)
- **Fast Performance** - Next.js 14 with optimized loading
- **Beautiful Design** - Warm, small-town aesthetic with modern colors

## 📋 Pages

- **Home** (`/`) - Main landing page with all key information
- **Services** (`/services`) - Detailed service offerings with pricing
- **Contact** (`/contact`) - Contact form, hours, and service areas

## 🎨 Design System

- **Colors**: Forest green, warm amber, clean whites
- **Typography**: Modern sans-serif with excellent readability
- **Layout**: Mobile-first responsive design
- **Spacing**: Consistent rhythm for visual hierarchy

## 🛠️ Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Language**: TypeScript

## 📦 Installation

npm install

## 🏃 Development

npm run dev

Visit http://localhost:3000 to see the site.

## 🚢 Deployment

npm run build
npm run start

Or deploy to Vercel (recommended for Next.js)
